# 🏦 Bank Loan Performance & Risk Analysis Dashboard

Power BI | SQL | Data Analytics | Financial Analytics | Business Intelligence

---

## 📌 Project Description

The **Bank Loan Performance & Risk Analysis Dashboard** is a data analytics project designed to evaluate and monitor the lending performance of a financial institution. The project focuses on transforming raw loan data into meaningful insights using **SQL for data analysis and Power BI for interactive dashboard development**.

This analytical solution enables stakeholders to gain visibility into loan portfolio performance, borrower characteristics, and repayment patterns. Through structured data modeling and effective visualizations, the dashboard provides a clear understanding of lending operations and helps support data-driven decision making.

The project demonstrates how **Business Intelligence tools can be applied in the financial domain to analyze loan data, identify patterns, and monitor portfolio health**.

---

# 📊 Project Overview

The objective of this project is to build a comprehensive reporting solution that analyzes bank loan data and provides insights into lending performance. The dashboard enables monitoring of loan applications, funding activity, repayment trends, and borrower characteristics across different dimensions.

The Power BI report is divided into **three dashboards**, each serving a specific analytical purpose:

- **Summary Dashboard** – Provides a high-level overview of lending performance.
- **Overview Dashboard** – Presents visual insights into loan distribution and borrower trends.
- **Details Dashboard** – Allows detailed exploration of loan-level records.

Together, these dashboards provide a structured and interactive view of the bank’s loan portfolio.

# 🗃 Dataset Details

**Total Rows:** ~38,000+  
**Total Columns:** 20+

### Column Description

| Column Name | Description |
|-------------|-------------|
| id | Unique identifier for each loan |
| issue_date | Date when the loan was issued |
| loan_amount | Total funded loan amount |
| total_payment | Total amount received from borrower |
| int_rate | Interest rate applied to the loan |
| dti | Debt-to-Income ratio of borrower |
| loan_status | Loan status (Current, Fully Paid, Charged Off) |
| term | Loan term (36 months / 60 months) |
| emp_length | Employment length of borrower |
| purpose | Purpose of the loan |
| home_ownership | Borrower home ownership status |
| address_state | State of the borrower |

# 📄 Problem Statement

Financial institutions handle large volumes of loan applications and must continuously monitor their lending performance to ensure a healthy and profitable loan portfolio. However, analyzing raw financial data without a structured reporting system makes it difficult to identify lending trends, borrower behavior, and potential risk factors.

The main objectives of this project are:

- To analyze the **total number of loan applications** received over a specific period.
- To evaluate the **total amount of funds disbursed as loans** and track lending activity.
- To monitor the **total amount received from borrowers**, helping assess repayment performance.
- To distinguish between **Good Loans and Bad Loans** based on loan status in order to evaluate the quality of the loan portfolio.
- To analyze **borrower financial indicators**, such as interest rate and debt-to-income ratio.
- To understand **loan distribution across different categories** including loan term, employment length, loan purpose, and home ownership status.
- To analyze **regional lending patterns** to identify states with higher loan activity.
- To develop an **interactive Power BI dashboard** that provides a comprehensive view of lending performance and supports data-driven decision making.

By addressing these objectives, the project aims to provide a clear analytical framework for evaluating loan portfolio health and identifying patterns within lending operations.
